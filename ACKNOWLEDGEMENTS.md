## Acknowledgements

These acknowledgements apply to the whole **PyDofus** organization, not only to this repository.

### Unity data extraction

- [K0lb3](https://github.com/K0lb3) — author of
  [UnityPy](https://github.com/K0lb3/UnityPy), the core library used to read
  and extract data from the dofus Unity client, and of
  [fpng-py](https://github.com/K0lb3/fpng_py), used as an optional fast PNG
  encoder.
- [LukeFZ](https://github.com/LukeFZ) — author of
  [Il2CppInspectorRedux](https://github.com/LukeFZ/Il2CppInspectorRedux), used
  both by our code generator and to produce the IL2CPP dumps (`dump.cs`,
  `dummyDll`, ...).
- [Nesrak1](https://github.com/nesrak1) — author of
  [USCSandbox](https://github.com/nesrak1/USCSandbox), used to recover shaders
  for map and sprite rendering, and of
  [AddressablesTools](https://github.com/nesrak1/AddressablesTools), which
  served as the reference for our Python catalog reader.

### FMOD audio extraction

- [Neptuwunium](https://github.com/neptuwunium) — author of
